Приложение "Школьное расписание звонков" - это программное обеспечение, предназначенное для автоматической, гибко настраеваемой, подачи звуковых сигналов по расписанию.

Возможности программы:

  • До 20-ти уроков в смене, включая нулевой урок;
  • Две смены;
  • Установка времени начала и окончания урока (периода);
  • Установка времени предупреждающего (за определенное время до начала урока) звукового сигнала;
  • 4 отдельных расписания;
  • Выбор расписания для каждого дня недели;
  • Ручная подача звукового сигнала;
  • Выбор мелодии для ручной подачи и предупреждающего звонка;
  • Выбор мелодии для начала и окончания каждого урока;
  • Возможность сохранения расписаний и всех настроек приложения в отдельный файл, а так же его открытие;
  • Выполнение сценариев (запуск приложений) операционной системы, перед подачей звукового сигнала и после его завершения;
  • TCP/IP сервер, для взаимодействия с приложением "Школьное расписание звонков - Монитор"

Программное обеспечение разрабатывалось для учебного учреждения. Но так же может использоваться в любой сфере, где необходимо использовать автоматическую подачу звуковых сигналов по расписанию.

Программа написана с использованием библиотеки Qt5, что делает её кроссплатформенной. Тестировалась на таких операционных системах как: Windows 7(x86, x64), Ubuntu 14.04 - 18.04(x86, x64), FreeBSD 10 - 11(x86, x64).
