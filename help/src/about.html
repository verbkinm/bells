О программе:

Данное программное обеспечение было написано для сервера школьных звонков. «Школьное расписание звонков» основано на Qt 5, Собрано в апреле 2016 года для Лицея им. Г.Ф. Атякшева. Тестировалось на Windows 7, Linux Ubuntu 14.04, FreeBSD 10.1.

Ссылка для скачивания - https://github.com/verbkinm/bells.
