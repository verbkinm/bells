Интерфейс приложения:

Панель «Меню»:

Панель «Меню» делится на четыре части — Файл, Звонок, Настройки, Справка.

Файл:
  • Применить( ) — сохраняет все внесённые изменения на вкладках расписаний и устанавливает рабочим расписанием текущую вкладку расписания.
  • Открыть шаблон( ) — диалоговое окно выбора ранее сохранён ного шаблона. При открытии шаблона полностью применяються все настройки - расписание, мелодии, пути к каталогам, программы запуска, дни недели и т.д.
  • Сохранить шаблон( ) — диалоговое окно выбора места для сохранения файла шаблона. Файл шаблона включает в себя все настройки программы - расписание, мелодии, пути к каталогам, программы запуска, дни недели и т.д.
  • Выход( ) — выход из программы и завершение всех запущенных программ основным приложением.
Звонок:
  • Подать звонок( ) — подаёт звуковй сигнал выбранной мелодии, которая устанавливается в Общих настройках. Во время воспроизведения меняется на Выключить звонок( ) , при нажатии на который воспроизведение мелодии прекращается.
Настройки:
  • Дни недели( ) — детальная настройка расписаний, времени и мелодий до начала урока, так же включение и выключение воспроизведения мелодий для каждого дня недели.
  • Общие( ) — общие настройки, включающие в себя: выбор каталога с мелодиями, мелодия для подачи звукового сигнала вручную, количество уроков на каждую смену в каждом расписании, выбор программ для запуска перед подачей звукового сигнала и после.
  • Дата и время( ) — настройка даты и времени в системе.
  • Язык( ) — выбор языка интерфейса программы.
Справка:
  • Инструкция( ) — вывод данной инструкции.
  • О программе( ) — информация о данном программном продукте и об авторе.
Панель инструментов:

Панель инструментов предназначена для быстрого выбора элементов панели «Меню». На панели размещены такие элементы: Применить, Открыть шаблон, Сохранить шаблон, Подать звонок, Дни недели, Общие, Дата и время, Инструкция, Выход. Панель инструментов можно отключить, нажав на ней, либо на панели «Меню», правой кнопкой и выбрать строку «панель инструментов». Для включения панели инструментов нужно нажать правой кнопкой на панель «Меню» и выбрать «панель инструментов».

Основная часть приложения:

Основная часть окна приложения делится на левую и правую панель. Размер панелей изменяется, если «схватить» мышкой границу этих панелей и переместить влево или вправо. В левой панели приложения выставляются данные о расписаниях, такие как время и мелодия начала и окончания уроков и т. д. Так же можно переименовать вкладку расписания щёлкнув по ней дважды.

В правой панели указываются данные о рабочем (установленном) расписании, согласно которым происходит запуск звукового сигнала в то или иное время. Существует две вкладки для смен, если смена выключена, то вкладка становится не активной. В верхней части правой панели после слова «Установлено:» написано название расписания (вкладки) из левой панели, согласно которому установлено рабочее расписание.

Панель текущей даты и времени:

На данной панели отображается дата, время и название дня недели, согласно установленным в системе настройкам.
